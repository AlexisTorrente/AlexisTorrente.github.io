Página web de Panaderia




Este proyecto fue la entrega final para las materias de Programación III y Laboratorio de computación III. Este proyecto es sobre la creación de una página web para ventas online para una panaderia "x" con dos tipos de usuarios: Empleado y Cliente (para más info sobre la funcionalidad de cada uno, ver el word en el repositorio). Se hizo tanto el frontend como el backend, utilizando la IDE Microsoft Visual Studio para el desarrollo de la página, y Microsoft SQL Server Management Studio para la creación de la base de datos con sus respectivas tablas, registros y procedimientos almacenados. Se utilizó el lenguaje C# con .NET framework y TSQL para la realización de éste proyecto.
